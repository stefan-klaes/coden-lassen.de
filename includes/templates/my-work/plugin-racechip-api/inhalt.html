Briefing:

  • Integration der Racechip API in Woocommerce
  • Auswahl der Fahrzeuge per Dropdwon - Hersteller, Modell, Motorisierung
  • Übernahme der Preise aus der Racechip API

Umsetzung

Ich habe für diesen Auftrag ein individuelles Plugin programmiert. Auf der Produktseite gab es Auswahlfelder für Hersteller, Marke und Motor. Die Auswahl der Fahrzeuge wurde über die Racechip API geladen.

Im Warenkorb wurden die entsprechenden Merkmale für das jeweilig gewählte Chiptuning angezeigt. Ebenso wurden die Merkmale im Woocommerce Backend angezeigt.

Zusätzlich wurde ein Shortcode bereitgestellt, der alle Daten anzeigte und nach der Auswahl zum entsprechenden Produkt linkte.
